The referee you can leave behind

An independent, deterministic checker for single-cell analyses. It doesn't give an opinion on your pipeline — it re-runs the statistically correct procedure itself and shows exactly where a reported result doesn't hold up.

check #1 · power-independent

Confounding — is the effect even estimable?

Pure design-matrix algebra, no statistics of power involved — and it is graded. Complete aliasing is rare; partial confounding is the norm. The tool regresses the condition indicator on the nuisance design matrix and returns two numbers: R²/VIF (can it be separated, and what does adjusting cost) and λ (how much of the run effect leaks in if you don't adjust). check #2 · earned by recompute

Pseudoreplication — cells aren't replicates

A per-cell test treats thousands of cells as independent samples, so tiny effects look wildly significant. The biological replicate is the donor. sc-referee aggregates each donor to one pseudobulk sample and re-runs the test. Press collapse.

The verdict is earned: a BLOCKER only fires when the recompute is adequately powered. At low n the same check honestly returns needs_evidence — it blocks when powered, abstains when not.

Six statuses, conservative by construction

A blocker fires only when the math unambiguously earns it. Everything uncertain stays advisory — a referee that false-accuses a valid analysis is worse than none. Hover a rung.

why not "just ask the AI"

Four things a prompt structurally can't be

Deterministic

Ask Claude twice, get two answers. sc-referee returns the same verdict every run — so you can gate CI on it.

Independent

The analysis was generated with an AI. Asking that same AI to grade it shares its blind spots. The verdict here comes from arithmetic, not the generator.

Encoded

It applies the expert checklist without the user knowing to ask. The rushed grad student never types "check for pseudoreplication." These are silent errors.

Unattended

It runs on every analysis, forever — as a CLI and a GitHub Action — including long after the resident bioinformatician has left.
