# GeneBench GB-P07 demo — attribution

This demo runs on the external **GeneBench-Pro GB-P07** benchmark (a donor-level eQTL task for CXCL10). The
benchmark's released bytes are **not redistributed** in this repository — only sc-referee's analysis
contract (`sc-referee-capsule.yaml`), the prepared structural proposal, and recorded provenance digests are
committed. To run the demo you supply your own copy of the benchmark archive (`GB-P07-data.zip`) and
materialize it locally (see this folder's `README.md`). No benchmark data and no held-out reference answers
are included here.
